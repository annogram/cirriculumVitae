ParaTracer

Parallel computing is very mature in lanuages such as C/C++ and Fortran and there are many well known methods of instrumentation. Java does have its own automated profiling and tracing framework from the Java Mission Control application that is now bundled with all Java JDKs. However the tracefiles created by this tool can only be used by this tool and is notably lacking in parallel program monitoring.

Paratracer is an instrumentation tool that provides the ability for developers to manually add classes into a multiprocessor Java program. Developers can create classes that ParaTracer can insert into their code at runtime.

  • More

Java, Parallel and High performance computing, runtime monitoring

Pinscape

Pinscape is a 2D platformer game that focusus on teamwork and problem solving to solve puzzles and help two pinballs trapped in their hellish prison escape from their torment to the outside world. Players can collect coins placed in challenging locations to increase their score and play an infinite survival mode with a procedurally generated landscape.

Pinscape was developed in Unity and all assets, physics and levels were designed by the development team. Players can unlock levels by completing previous levels on lower difficulties and when they have completed the game they can challenge other players scores online with the leaderboard feature.
